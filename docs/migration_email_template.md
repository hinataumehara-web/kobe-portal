# 移行告知メール 文面テンプレ

## 件名

```
【食料環境経済学コース 学生ポータル】次回ログイン時にパスフレーズの設定をお願いします
```

## 本文(プレーンテキスト)

```
食料環境経済学コースのみなさま

お疲れさまです。学生ポータルの運営をしている梅原ひなたです。

セキュリティ強化のため、登録されている氏名・学籍番号・履修科目・成績を
ブラウザ側で暗号化してから保存する方式に切り替えました。

これにより、

  ・サーバー管理者(私を含む)からも内容が見えなくなる
  ・万一データが流出しても暗号文しか読めない

という保護がかかります。

【お願い】
次回ログイン時に「パスフレーズの設定」画面が一度だけ表示されます。
画面の指示に従って、以下を設定してください。

  1. 自分だけが知っているパスフレーズ(10文字以上)を決める
  2. 表示される「リカバリーコード」を必ず保存する
     (パスフレーズを忘れたときの唯一の救済手段です)

設定が完了すると、以後は同じブラウザでは自動的にログインされ
パスフレーズを毎回入力する必要はありません。

【ポータル URL】
https://(あなたのドメインを入れる)

【ご注意】
・パスフレーズはサーバーに送信・保存されません
・パスフレーズとリカバリーコードの両方を失うと、履修データを
  取り戻す手段がなくなります
・リカバリーコードは画面に表示されたタイミングでのみ表示されます。
  必ずダウンロードまたはコピーして保管してください

不明点があればこのメールに返信してください。

ーー
食料環境経済学コース 学生ポータル
梅原 ひなた
```
